My Story

Hi! I'm Elan Miles, a passionate programmer who loves creating efficient solutions and exploring new technologies. My journey in software development began with a fascination for how code can transform ideas into reality.

I specialize in Python and C# development, creating everything from web applications to system utilities. My favorite hobby is programming itself - there's something magical about solving problems through code and seeing applications come to life.

Currently, I'm expanding my expertise into C++ and exploring modern development frameworks. Previously, I worked with PHP in Devel Studio, which gave me a solid foundation in web development before transitioning to more popular and powerful languages.

What drives me is the constant evolution of technology and the endless possibilities it offers. Every project is a new opportunity to learn, grow, and create something that can make a difference.
